# Data security
## Securing data in storage

### Threats
- Hackers
- Malicious insiders
- Careless insiders
- Industrial espionage
- Terrorism and saboteurs
- Natural disasters
- Hardware failures
- Power failure
- Data leakage

### Risks
- These threats usually map to one of the three risks
    1. Data and systems becoming unavailable
    2. Data getting corrupt
    3. Data Falling into wrong hands


### Securing data
1. Data in storage
    - Databases
    - Storage media
    - Files
    - Code
2. Data in transit
    - Data being transmitted over network

### Data in storage: risk examples
- Hard drive crashes and we lose all our application data
- Careless employee accidentaly deletes important data
- Hacker accesses database and steal credentials

### How to protect data in storage
- Physical security (secure data center for example)
- Access control
- Seperation of duties
- Encryption
- Regular patching and updating software and servers
- Backups
- Disaster recovery plan
- Data redundancy
